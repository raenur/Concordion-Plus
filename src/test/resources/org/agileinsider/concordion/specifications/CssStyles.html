CSS Styles

Example

When the Concordion Plus Extension is installed that adds the scenario and ignore commands in the http://www.agileinsider.org/concordion/plus namespace. The concordion-plus style sheet should be included.

Running a specification containing: 

    <div xmlns:cp="http://www.agileinsider.org/concordion/plus" xmlns:c="http://www.concordion.org/2007/concordion">
        <div cp:scenario="Simple Scenario">Inside Scenario</div>
    </div>

Should add the following styles:

  • .pass
  • .fail
  • .error
  • .skip
