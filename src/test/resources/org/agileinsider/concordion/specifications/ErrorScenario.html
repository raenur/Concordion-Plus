Error Scenario

Example

When the Scenario Extension is installed that adds the scenario command in the http://www.agileinsider.org/concordion/plus namespace. This command should highlight error scenarios.

Running a specification containing: 

<div xmlns:cp="http://www.agileinsider.org/concordion/plus" xmlns:c="http://www.concordion.org/2007/concordion">
    <p cp:scenario="Error Scenario">
        Should get an <span c:execute="nonExistingMethod()">Exception</span>
    </p>
</div>

Should match:

<div xmlns:cp="http://www.agileinsider.org/concordion/plus" xmlns:c="http://www.concordion.org/2007/concordion">
    <p cp:scenario="Error Scenario" class="error">
        .*<span class="exceptionMessage">.*
    </p>
</div>
