Concordion Plus Specifications

  • CSS Styles should be included in the outputs.
  • Successful scenarios should result in the scenario being classified as "pass".
  • Failing scenarios should result in the scenario being classified as "fail".
  • Errors in scenarios should result in the scenario being classified as "error".
  • Ignored sections should result in the scenario being classified as "skip".
  • Should be able to process multiple scenarios.
  • After encountering an Ignored section other scenarios should still run.
  • After encountering a Failing scenario other scenarios should still run.
  • After encountering an Error in a scenario other scenarios should still run.
