Failing Scenario

Example

When the Scenario Extension is installed that adds the scenario command in the http://www.agileinsider.org/concordion/plus namespace. This command should highlight failing scenarios.

Running a specification containing: 

<div xmlns:cp="http://www.agileinsider.org/concordion/plus" xmlns:c="http://www.concordion.org/2007/concordion">
    <p cp:scenario="Failing Scenario">
        Should get a <span c:assertEquals="getText()">failure</span>
    </p>
</div>

Should produce:

<div xmlns:cp="http://www.agileinsider.org/concordion/plus" xmlns:c="http://www.concordion.org/2007/concordion">
    <p cp:scenario="Failing Scenario" class="fail">
        Should get a <span c:assertEquals="getText()" class="failure"><del class="expected">failure</del>
<ins class="actual">unexpected text</ins></span>
    </p>
</div>
