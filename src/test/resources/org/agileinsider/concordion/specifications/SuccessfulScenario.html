Successful Scenario

Example

When the Scenario extension is installed that adds the scenario command in the http://www.agileinsider.org/concordion/plus namespace. This command will highlight passing scenarios.

Running a specification containing: 

<div xmlns:cp="http://www.agileinsider.org/concordion/plus" xmlns:c="http://www.concordion.org/2007/concordion">
    <div cp:scenario="Simple Scenario">Inside Scenario</div>
</div>

Should produce:

<div xmlns:cp="http://www.agileinsider.org/concordion/plus" xmlns:c="http://www.concordion.org/2007/concordion">
    <div cp:scenario="Simple Scenario" class="pass">Inside Scenario</div>
</div>
